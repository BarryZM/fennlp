# fennlp

An out-of-the-box NLP toolkit can easily help you solve tasks such as entity recognition, relationship extraction, text classfication and so on.

# Require
tensorflow>=2.0

# Usage
1. clone source
```
git clone
```
2. install package
```
python setup.py install
```
3.run test file
```
cd tests
```
```
python bert_ner_train.py
```
